## 📄 论文摘要

快递物流网络的快速扩张深刻改变了城市消费空间格局。本文基于1352份有效问卷，综合运用**描述性统计、二元Logistic回归和条件Logit效用函数**，系统考察快递取件行为对附带消费的诱发效应。

**核心发现：**
- 📊 **85.5%** 的受访者取快递时产生附带消费意愿
- 👥 工作人群是核心驱动力 (**OR = 13.58, p < 0.001**)
- 🍔 餐饮消费占比**93.3%**，为绝对主力
- 📍 距离贡献驿站选择总效用的**25.35%**
- 🏪 **84.1%** 受访者表示便利设施会提高消费意愿

## 🔬 计量方法

### 二元 Logistic 回归

对二分类因变量（是否消费 / 是否增加频率 / 设施是否有效）使用极大似然估计：

$$L(\beta) = \prod_{i=1}^{n} p_i^{y_i} (1-p_i)^{1-y_i}, \quad p_i = \frac{1}{1+e^{-X_i\beta}}$$

报告系数、标准误（Hessian矩阵逆）、z值、p值、优势比（OR）及95%置信区间。

### 条件 Logit 效用函数

基于 McFadden (1974) 的随机效用框架：

$$U_{ij} = V_{ij} + \varepsilon_{ij}$$

消费者在5个驿站特征中选择效用最大者，排序数据用于估计各因素的边际效用贡献。

## 📖 研究设计

- **调查方式**：线上问卷（微信平台）+ 线下发放
- **调查时间**：2025年1月22日 - 2月8日
- **覆盖范围**：全国31个省（自治区、直辖市）
- **样本量**：1352份有效问卷（Q7注意力检验通过率84.8%）
- **问卷结构**：10道核心题目，含排序题、多选题、二元选择题
